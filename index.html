상품코드 붙여넣기 → 변환 → 한 번에 검색

1. 입력 · 상품코드 붙여넣기

줄바꿈·탭·쉼표 섞여 있어도 괜찮아요.
· 시트/SCM에서 여러 셀을 그대로 복사해서 붙여넣어도 됩니다.
· 코드가 서로 붙어있으면 하나의 코드로 인식합니다.

2. 결과 · 쉼표(,) 구분 문자열

검색창에 그대로 붙여넣어 사용하세요.
· [결과 복사하기] 버튼으로 바로 클립보드에 복사할 수 있습니다.
· 동작 방식
  • 공백(스페이스·탭·줄바꿈)과 쉼표(,)를 모두 구분자로 인식합니다.
  • 빈 값은 자동으로 제거하고, 코드만 연결해서 깔끔하게 정리합니다.
  • 중복 코드도 그대로 유지합니다. (중복 제거 기능이 필요하면 추후 추가 가능)
